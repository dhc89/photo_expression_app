# 概要
「photo expression」というアプリケーションで、撮った写真を表情ごとに登録できます。


# 想定しているユーザー
・写真をよく撮る方、見る方
 
・子どもの成長を写真で見たいご家族

・SNS用の投稿写真をすぐ探したい方

# 詳細
・どの表情で写真を撮ったかを一目で確認できます

・お気に入りに投稿できます

# 作成背景
・実際の現場を想定したかったため、自分の興味だけでなく、「ユーザーからの意見を聞き作ること」を考え、妻、両親からのヒアリングをもとに本アプリを構想しました。
 
・私の妻は息子(1歳)の写真をよく撮ります。それを見ている両親から、「孫の沢山の写真から特定の表情の写真がふと見たくなる」という意見をヒアリングしました。
　そこで、表情ごとに写真を管理できたら、探す手間がなく、「すぐ見れたらいいな！」との思いから、制作しました。

# ユーザーへの思い
・ご家族の方々が特定の表情を探す手間がなく、すぐに見れて、「子供、孫の成長を感じて、嬉しい気持ちになってほしい」と思い、制作しました。

・子供だけでなく、友人、恋人など写真を撮る方に全ての方が使い、思い出をすぐ見返せて、「懐かしい気持ち、楽しい気持ちになってほしい」と思い、制作しました

・SNS用の投稿写真をすぐ探せることで「楽に探してほしい」と思い、制作しました。

# 工夫した点
・表情ごとに保存できる。

・表情ごとに分けるボタンを表情ごとのアイコンを使用。

・ヒアリングをもとに作成、アプリを実際に使っていただき、フィードバックをもらいました。(ユ -ザ-視点での開発)
